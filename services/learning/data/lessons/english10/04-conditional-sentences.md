# Conditional Sentences - Câu điều kiện

## 1. Conditional Type 0 (Điều kiện loại 0)

### 1.1. Cấu trúc
**If + S + V(present simple), S + V(present simple)**

### 1.2. Cách dùng
- Diễn tả sự thật hiển nhiên, chân lý
- Diễn tả thói quen, điều luôn đúng

### 1.3. Ví dụ
- If you **heat** ice, it **melts**. (chân lý)
- If you **mix** red and blue, you **get** purple.
- If I **wake up** late, I **miss** the bus. (thói quen)
- Water **boils** if you **heat** it to 100°C.

### 1.4. Lưu ý
- Có thể thay "if" bằng "when"
- When you heat ice, it melts.

## 2. Conditional Type 1 (Điều kiện loại 1)

### 2.1. Cấu trúc
**If + S + V(present simple), S + will + V(infinitive)**

### 2.2. Cách dùng
- Diễn tả điều kiện có thể xảy ra ở hiện tại hoặc tương lai
- Điều kiện có thật, có khả năng thực hiện được

### 2.3. Ví dụ
- If it **rains** tomorrow, I **will stay** at home.
- If you **study** hard, you **will pass** the exam.
- If she **doesn't hurry**, she **will miss** the train.
- If they **invite** me, I **will go** to the party.

### 2.4. Các biến thể
- Mệnh đề chính có thể dùng: can, may, must, should
  - If you finish early, you **can** go home.
  - If it rains, you **should** take an umbrella.
- Mệnh đề "if" có thể đứng sau
  - I will help you if you need.

## 3. Conditional Type 2 (Điều kiện loại 2)

### 3.1. Cấu trúc
**If + S + V(past simple), S + would/could/might + V(infinitive)**

### 3.2. Cách dùng
- Diễn tả điều kiện không có thật ở hiện tại
- Diễn tả điều kiện trái với sự thật hiện tại
- Lời khuyên, ước muốn

### 3.3. Ví dụ
- If I **had** a lot of money, I **would buy** a big house. (không có nhiều tiền)
- If I **were** you, I **would study** harder. (lời khuyên)
- If she **knew** his address, she **would visit** him. (không biết địa chỉ)
- If it **didn't rain**, we **could go** out. (trời đang mưa)

### 3.4. Lưu ý đặc biệt
- Với động từ "be", dùng "were" cho tất cả các ngôi (không dùng "was")
  - If I **were** rich... ✓
  - If I was rich... ✗ (không chuẩn)
- If I were you = nếu tôi là bạn (lời khuyên)

## 4. Conditional Type 3 (Điều kiện loại 3)

### 4.1. Cấu trúc
**If + S + had + V3/ed, S + would/could/might + have + V3/ed**

### 4.2. Cách dùng
- Diễn tả điều kiện không có thật trong quá khứ
- Hối tiếc về điều đã không xảy ra trong quá khứ

### 4.3. Ví dụ
- If I **had studied** harder, I **would have passed** the exam. (không học chăm → trượt)
- If she **had known** earlier, she **would have come**. (không biết sớm → không đến)
- If they **had left** earlier, they **wouldn't have missed** the train. (không đi sớm → lỡ tàu)
- If it **hadn't rained**, we **could have gone** out. (trời mưa rồi → không đi được)

### 4.4. Ý nghĩa
Dùng để diễn tả sự hối tiếc, nuối tiếc về quá khứ:
- If I had known, I would have helped you. (tiếc là không biết)

## 5. So sánh các loại câu điều kiện

| Loại | Điều kiện | Kết quả | Ví dụ |
|------|-----------|---------|-------|
| **Type 0** | Luôn đúng | Luôn đúng | If you heat water, it boils. |
| **Type 1** | Có thể xảy ra (hiện tại/tương lai) | Có thể xảy ra | If it rains, I will stay home. |
| **Type 2** | Không có thật (hiện tại) | Không có thật | If I were rich, I would travel. |
| **Type 3** | Không có thật (quá khứ) | Không có thật | If I had known, I would have come. |

## 6. Mixed Conditionals (Câu điều kiện hỗn hợp)

### 6.1. Type 3 + Type 2
Điều kiện trong quá khứ, kết quả ở hiện tại

**If + S + had + V3/ed, S + would + V(infinitive)**

**Ví dụ:**
- If I **had studied** medicine, I **would be** a doctor now.
  (không học y → bây giờ không phải bác sĩ)

### 6.2. Type 2 + Type 3
Điều kiện ở hiện tại, kết quả trong quá khứ

**If + S + V(past), S + would have + V3/ed**

**Ví dụ:**
- If I **were** more careful, I **wouldn't have made** that mistake.
  (không cẩn thận → đã mắc lỗi)

## 7. Unless (trừ khi)

### 7.1. Cách dùng
**Unless = If... not**

### 7.2. Ví dụ
- Unless you study, you will fail. = If you don't study, you will fail.
- I won't go unless you come. = I won't go if you don't come.
- Unless it rains, we will go out. = If it doesn't rain, we will go out.

## 8. Inversion in Conditionals (Đảo ngữ trong câu điều kiện)

### 8.1. Type 1
Were/Should + S + (to) V, S + would...

- Should you need help, call me. = If you should need help, call me.

### 8.2. Type 2
Were + S + (to) V, S + would...

- Were I you, I would study harder. = If I were you, I would study harder.

### 8.3. Type 3
Had + S + V3/ed, S + would have...

- Had I known, I would have come. = If I had known, I would have come.

## 9. Bài tập

### Exercise 1: Xác định loại câu điều kiện
1. If you heat ice, it melts.
2. If I had a car, I would drive to work.
3. If she studies hard, she will pass.
4. If they had left earlier, they wouldn't have missed the train.

### Exercise 2: Chia động từ
1. If I __________ (be) you, I would accept the job.
2. If she __________ (study) harder, she would have passed.
3. If it __________ (rain) tomorrow, we will cancel the trip.
4. If he __________ (know) the answer, he would tell you.
5. If they __________ (invite) me, I will go.

### Exercise 3: Viết lại câu
1. I don't have money, so I can't buy it.
   → If I __________, I __________.

2. She didn't study, so she failed.
   → If she __________, she __________.

3. You don't hurry, so you will be late.
   → If you __________, you __________.

4. He didn't know, so he didn't help.
   → If he __________, he __________.

### Exercise 4: Chọn đáp án đúng
1. If I _____ you, I would apologize.
   - A. am   B. was   C. were   D. be

2. If she _____ harder, she would have passed the exam.
   - A. studied   B. had studied   C. studies   D. has studied

3. If it _____ tomorrow, we will stay at home.
   - A. rain   B. rains   C. rained   D. will rain

### Exercise 5: Viết câu điều kiện
1. I am not rich. I can't buy a car. (Type 2)
2. She didn't invite me. I didn't go to the party. (Type 3)
3. You study hard. You will pass the exam. (Type 1)
4. He doesn't know English. He can't get the job. (Type 2)
5. They left late. They missed the train. (Type 3)
