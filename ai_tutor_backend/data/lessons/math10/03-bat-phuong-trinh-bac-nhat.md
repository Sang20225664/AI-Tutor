# Bất phương trình bậc nhất một ẩn

## 1. Định nghĩa

Bất phương trình bậc nhất một ẩn là bất phương trình có một trong các dạng:

- **ax + b > 0**
- **ax + b ≥ 0**
- **ax + b < 0**
- **ax + b ≤ 0**

Trong đó: **a, b** là các số đã biết, **a ≠ 0**, và **x** là ẩn số.

Nghiệm của bất phương trình là giá trị của x làm cho bất phương trình đúng. Tập hợp tất cả các nghiệm gọi là **tập nghiệm** của bất phương trình.

## 2. Tính chất của bất phương trình

### Tính chất 1: Cộng (trừ) cùng một số

Khi cộng (hoặc trừ) cả hai vế của bất phương trình với cùng một số, ta được bất phương trình **tương đương** với bất phương trình đã cho.

**Ví dụ:**
- x + 3 > 5 ⟺ x > 5 - 3 ⟺ x > 2
- x - 2 < 7 ⟺ x < 7 + 2 ⟺ x < 9

### Tính chất 2: Nhân (chia) với số dương

Khi nhân (hoặc chia) cả hai vế của bất phương trình với cùng một **số dương**, ta được bất phương trình tương đương với bất phương trình đã cho, **chiều bất phương trình không đổi**.

**Ví dụ:**
- 2x > 6 ⟺ x > 3 (chia hai vế cho 2)
- x/3 < 4 ⟺ x < 12 (nhân hai vế với 3)

### Tính chất 3: Nhân (chia) với số âm

Khi nhân (hoặc chia) cả hai vế của bất phương trình với cùng một **số âm**, ta được bất phương trình tương đương với bất phương trình đã cho, nhưng phải **đổi chiều** bất phương trình.

**Ví dụ:**
- -2x > 4 ⟺ x < -2 (chia hai vế cho -2 và đổi chiều)
- -x/3 ≤ 2 ⟺ x ≥ -6 (nhân hai vế với -3 và đổi chiều)

## 3. Cách giải bất phương trình bậc nhất

### Giải bất phương trình ax + b > 0

**Bước 1**: Chuyển vế số hạng tự do
- ax > -b

**Bước 2**: Chia hai vế cho a

- Nếu **a > 0**: x > -b/a (giữ nguyên chiều)
- Nếu **a < 0**: x < -b/a (đổi chiều)

Tương tự cho các dạng bất phương trình khác (≥, <, ≤).

## 4. Ví dụ minh họa

### Ví dụ 1: Hệ số a dương

Giải bất phương trình: 3x - 6 > 0

**Giải:**
```
3x - 6 > 0
3x > 6        (chuyển vế)
x > 2         (chia hai vế cho 3, giữ nguyên chiều vì 3 > 0)
```

Vậy tập nghiệm: S = {x | x > 2} = (2; +∞)

### Ví dụ 2: Hệ số a âm

Giải bất phương trình: -2x + 8 ≤ 0

**Giải:**
```
-2x + 8 ≤ 0
-2x ≤ -8      (chuyển vế)
x ≥ 4         (chia hai vế cho -2, đổi chiều vì -2 < 0)
```

Vậy tập nghiệm: S = {x | x ≥ 4} = [4; +∞)

### Ví dụ 3: Bất phương trình có phân số

Giải bất phương trình: (2/3)x - 4 < 0

**Giải:**
```
(2/3)x - 4 < 0
(2/3)x < 4
x < 4 : (2/3)
x < 4 · (3/2)
x < 6
```

Vậy tập nghiệm: S = {x | x < 6} = (-∞; 6)

### Ví dụ 4: Đưa về dạng cơ bản

Giải bất phương trình: 5x + 3 ≥ 2x - 6

**Giải:**
```
5x + 3 ≥ 2x - 6
5x - 2x ≥ -6 - 3    (chuyển vế)
3x ≥ -9
x ≥ -3              (chia hai vế cho 3)
```

Vậy tập nghiệm: S = {x | x ≥ -3} = [-3; +∞)

### Ví dụ 5: Có dấu ngoặc

Giải bất phương trình: 2(x - 1) < 3(x + 2)

**Giải:**
```
2(x - 1) < 3(x + 2)
2x - 2 < 3x + 6     (phá ngoặc)
2x - 3x < 6 + 2     (chuyển vế)
-x < 8
x > -8              (chia hai vế cho -1, đổi chiều)
```

Vậy tập nghiệm: S = {x | x > -8} = (-8; +∞)

## 5. Hệ bất phương trình

Hệ bất phương trình là hệ gồm hai hoặc nhiều bất phương trình cùng ẩn số.

### Cách giải hệ bất phương trình

**Bước 1**: Giải từng bất phương trình trong hệ

**Bước 2**: Lấy giao của các tập nghiệm

### Ví dụ 6: Giải hệ bất phương trình

Giải hệ bất phương trình:
```
{ x + 2 > 0       (1)
{ 3x - 6 < 0      (2)
```

**Giải:**

Giải (1):
```
x + 2 > 0
x > -2
```

Giải (2):
```
3x - 6 < 0
3x < 6
x < 2
```

Tập nghiệm của hệ: S = (-2; +∞) ∩ (-∞; 2) = (-2; 2)

Vậy nghiệm của hệ là: **-2 < x < 2**

## 6. Biểu diễn tập nghiệm trên trục số

### Quy ước:

- **Dấu ngoặc tròn (** hoặc **)**: không lấy điểm đó (< hoặc >)
- **Dấu ngoặc vuông [** hoặc **]**: lấy điểm đó (≤ hoặc ≥)

**Ví dụ:**

1. x > 2: Biểu diễn từ 2 đến +∞, dùng ngoặc tròn tại 2
2. x ≤ 5: Biểu diễn từ -∞ đến 5, dùng ngoặc vuông tại 5
3. -1 < x ≤ 3: Biểu diễn từ -1 đến 3, ngoặc tròn tại -1, ngoặc vuông tại 3

## 7. Bài toán thực tế

### Bài toán 1: Tìm giá trị

Tìm các giá trị của x để giá trị của biểu thức 3x - 5 lớn hơn giá trị của biểu thức x + 1.

**Giải:**

Ta có bất phương trình: 3x - 5 > x + 1
```
3x - x > 1 + 5
2x > 6
x > 3
```

Vậy x > 3.

### Bài toán 2: Tìm số tự nhiên

Tìm các số tự nhiên x thỏa mãn: 2x - 3 < 7

**Giải:**
```
2x - 3 < 7
2x < 10
x < 5
```

Vì x là số tự nhiên và x < 5, nên: x ∈ {0, 1, 2, 3, 4}

### Bài toán 3: Bài toán kinh tế

Một cửa hàng bán áo sơ mi. Giá mua vào là 150.000 đồng/cái, giá bán ra là 200.000 đồng/cái. Chi phí cố định hàng tháng là 5.000.000 đồng. Hỏi phải bán ít nhất bao nhiêu cái áo để có lãi?

**Giải:**

Gọi x (cái) là số áo bán được (x ∈ ℕ*, x > 0)

Doanh thu: 200.000x (đồng)

Chi phí: 150.000x + 5.000.000 (đồng)

Để có lãi: 200.000x > 150.000x + 5.000.000
```
200.000x - 150.000x > 5.000.000
50.000x > 5.000.000
x > 100
```

Vậy phải bán ít nhất 101 cái áo mới có lãi.

## 8. Các lỗi thường gặp

1. **Quên đổi chiều** khi nhân/chia với số âm
   - Sai: -2x > 4 ⟹ x > -2
   - Đúng: -2x > 4 ⟹ x < -2

2. **Nhầm lẫn giữa các ký hiệu**
   - > (lớn hơn) ≠ ≥ (lớn hơn hoặc bằng)
   - < (nhỏ hơn) ≠ ≤ (nhỏ hơn hoặc bằng)

3. **Quên điều kiện** trong bài toán thực tế
   - Ví dụ: số tự nhiên, số nguyên dương, etc.
